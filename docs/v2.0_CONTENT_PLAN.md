### 2.1 性格测试扩充题（30题）

> 特点：贴近大学生活场景，每道题4个选项各自对应独立结果

#### 性格题 18-25：室友相处场景

---

**【性格题-18】周末室友都在睡懒觉，但你有事要出门，你会？**

- **A. 设好闹钟轻手轻脚收拾，尽量不吵醒大家**  
  → 结果「温柔室友」：你是那种宁可自己麻烦也不想打扰别人的人。室友们可能从没发现过你悄悄出门的体贴，但你的细心一直在守护着这个小小的家。🌸

- **B. 直接开灯收拾，如果吵醒了就说声抱歉**  
  → 结果「坦荡室友」：你不愿为了迁就而委屈自己，但也不刻意制造矛盾。有什么事直接说，室友关系反而更简单纯粹。💫

- **C. 提前一晚把东西收拾好，早上直接走**  
  → 结果「计划型室友」：你擅长提前规划，把细节都想在前面。和你做室友最大的感受就是——永远不用担心意外，因为意外也在你的计划里。📋

- **D. 轻轻叫醒一个室友拜托帮忙递东西**  
  → 结果「社交型室友」：你懂得适时麻烦别人，反而拉近了距离。室友之间本就该互相帮忙，你让这种互助变得自然而然。🤝

---

**【性格题-19】室友半夜12点还在大声打游戏，你第二天有早课，你会？**

- **A. 发微信提醒他小点声，然后戴耳塞睡觉**  
  → 结果「隐忍型」：你习惯先自己消化问题，不愿意正面冲突。但记得偶尔也要表达真实需求，一味隐忍久了会委屈自己。😴

- **B. 直接起身说"我要睡了，麻烦小点声"**  
  → 结果「直接型」：你敢于表达自己的边界，不喜欢憋着。室友可能一时不适应，但久了反而会感谢你的坦诚。💪

- **C. 默默记下，明天找个机会委婉提一下**  
  → 结果「策略型」：你懂得时机很重要，不在气头上说事。处理问题讲究方式方法，让人感觉被尊重。🧠

- **D. 加入他们一起玩，早课的事明天再说**  
  → 结果「随性型」：你的人生哲学是——开心就好。有时候放过自己也是对的，但小心别让"明天再说"变成常态哦。🎮

---

**【性格题-20】室友把你的零食吃光了还毫无愧意，你会？**

- **A. 也吃掉他的零食，扯平**  
  → 结果「以牙还牙型」：你不愿吃亏，但也懂得拿捏分寸。以彼之道还施彼身，既不出气也不伤和气，刚刚好。⚖️

- **B. 认真跟他谈一次，说清楚你的感受**  
  → 结果「沟通型」：你相信话说开了问题就解决了一半。室友可能真的没意识到，你的提醒对他很重要。🗣️

- **C. 默默把零食藏到柜子深处，以后不分享了**  
  → 结果「自我保护型」：你习惯从源头上避免问题。但小心这种"冷处理"久了会让自己变得封闭哦。🔒

- **D. 下次直接买超大量，够大家一起吃**  
  → 结果「大气型」：与其计较，不如大方。你的逻辑是——既然要分享就分享个痛快，这点小事不值得费心。🎁

---

**【性格题-21】室友总是不经过同意就用你的东西（洗面奶/纸巾/充电宝），你会？**

- **A. 默默忍受，觉得用一点也无所谓**  
  → 结果「包容型」：你是那种"差不多就行"的人，不喜欢斤斤计较。但长期下来可能积累怨气，记得偶尔也要说"不"。🫠

- **B. 明确告诉ta以后要用先说一声**  
  → 结果「边界感型」：你很清楚自己的底线在哪，懂得保护自己的空间。温和但坚定地表达，让室友学会尊重你。🚧

- **C. 故意把东西藏起来，看ta什么反应**  
  → 结果「试探型」：你想通过观察反应来判断对方是否故意。但这种方法容易产生误会，不如直接沟通效率高。🔍

- **D. 自己也用ta的东西，建立"互通有无"规则**  
  → 结果「规则型」：你擅长建立双赢规则。与其一方单方面付出，不如定个大家都能接受的规矩。🤝

---

**【性格题-22】室友在准备考研/考公，你还在"躺平"，你会有什么感受？**

- **A. 有点焦虑，开始反思自己是不是太不努力了**  
  → 结果「自我反思型」：你容易被环境带着走，有竞争意识。但别让焦虑控制你，每个人时区不同。⏰

- **B. 完全不受影响，继续按自己的节奏生活**  
  → 结果「定力型」：你很清楚自己想要什么，不会被别人的节奏带偏。这种笃定是难得的品质。🎯

- **C. 想努力但又提不起劲，内心有点矛盾**  
  → 结果「纠结型」：你其实有上进心，只是缺少一点推力。不妨从小目标开始，慢慢找回状态。💭

- **D. 主动跟室友请教，看看能不能一起学习**  
  → 结果「行动型」：你懂得借力，把室友当成资源而非压力来源。一起进步是最好的相处模式。📚

---

**【性格题-23】期末考试周，室友找你聊天吐槽，你会？**

- **A. 先放下书陪ta聊完，反正复习效率也不高**  
  → 结果「陪伴型」：你把关系看得比任务重。室友找你倾诉是因为信任你，这份信任比复习重要。💕

- **B. 告诉ta等考完再聊，现在先各自复习**  
  → 结果「理性型」：你分得清轻重，知道什么时候该专注。室友可能一时失落，但会理解你的决定。📖

- **C. 一边聊天一边看书，两不耽误**  
  → 结果「多线程型」：你擅长同时处理多件事。但小心哪边都没处理好的情况哦。⚡

- **D. 约个时间专门聊，现在先帮ta整理一下复习重点**  
  → 结果「高效型」：你既想帮忙又想不影响复习，用最聪明的方式平衡了双方需求。🌟

---

**【性格题-24】第一次住校，想家了想哭一场，你会？**

- **A. 偷偷躲在被子里哭，不想让室友发现**  
  → 结果「内敛型」：你想自己消化情绪，不想给别人添麻烦。但有时候展示脆弱反而能拉近距离。🌙

- **B. 直接跟室友说"我想家了"，说完反而轻松了**  
  → 结果「坦诚型」：你把情绪当成正常的事，不觉得"想家"是软弱。室友会因此更懂你。💬

- **C. 找爸妈或老朋友打电话倾诉**  
  → 结果「独立型」：你习惯向家人和老朋友寻求支持，这是很好的减压方式。📞

- **D. 化思念为食欲，去食堂吃顿好的治愈自己**  
  → 结果「自愈型」：你懂得用行动调节情绪。没有什么是一顿美食解决不了的，如果有就两顿。🍜

---

**【性格题-25】室友的生活习惯和你差异很大（早睡/晚睡/洁癖/随意），你会？**

- **A. 制定"宿舍公约"，把规则白纸黑字写清楚**  
  → 结果「规则型」：你相信"丑话说前头"能避免大多数矛盾。提前约定比事后撕逼高效一百倍。📝

- **B. 各自保持习惯，互相不干涉**  
  → 结果「独立型」：你接受"和而不同"，不需要室友和自己一模一样。边界清晰反而更舒服。🤫

- **C. 主动调整自己去适应对方**  
  → 结果「适应型」：你愿意为关系做调整和牺牲。但注意别委屈自己太多哦。🤝

- **D. 观察一段时间，实在不行再想办法**  
  → 结果「观察型」：你喜欢先看再做判断，不急于下结论。有时候磨合需要时间。🔭

#### 性格题 26-30：社团/社交场景

---

**【性格题-26】社团面试被刷了，你会？**

- **A. 难过一阵子，然后认真分析自己哪里不足**  
  → 结果「复盘型」：你把失败当成学习机会。面试被刷不丢人，不知道为什么被刷才丢人。📊

- **B. 安慰自己"这个社团也不适合我"，很快释怀**  
  → 结果「心态型」：你擅长给自己找台阶，不让挫折影响心情太久。有时候放过自己很重要。🌈

- **C. 找社团里的人问问原因，为下次积累经验**  
  → 结果「行动型」：你把"为什么"看得比"结果"重要。主动问需要勇气，但这种勇气会带你走很远。💪

- **D. 积极投其他社团，不在一个坑里纠结**  
  → 结果「向前看型」：你不会被一个挫折绊住脚步。东边不亮西边亮，选择本来就很多。🚀

---

**【性格题-27】社团里有一个能力不如你但很会表现自己的同学，你会？**

- **A. 默默做好自己的事，让成果说话**  
  → 结果「实力型」：你相信是金子总会发光。与其争一时口舌，不如用实力证明一切。🏆

- **B. 有点不服气，也开始更积极地表现自己**  
  → 结果「竞争型」：你有胜负欲，这是好事。但注意别让竞争破坏了合作关系。⚔️

- **C. 反思是不是自己表达方式有问题**  
  → 结果「自省型」：你懂得从自己身上找原因。但有时候不是你的问题，只是风格不同而已。🤔

- **D. 主动和他合作，互相学习各自的长处**  
  → 结果「合作型」：你会欣赏别人的长处，而不是只盯着短处。取长补短是最聪明的策略。🤝

---

**【性格题-28】朋友找你帮忙做一件你其实不太情愿的事，你会？**

- **A. 勉强答应，事后又后悔**  
  → 结果「难拒绝型」：你不太会说不，宁可委屈自己也不愿让别人失望。但长期这样会让自己很累。😔

- **B. 直接说不，解释清楚原因**  
  → 结果「边界型」：你很清楚自己的底线，敢于拒绝。朋友如果因为这个生气，那也不是真朋友。🚫

- **C. 找一个合理的借口婉拒**  
  → 结果「委婉型」：你不想伤对方面子，又想保护自己。这种方式需要技巧，用多了容易被识破。🎭

- **D. 答应帮忙但提出条件（比如"下次你也要帮我"）**  
  → 结果「交换型」：你相信等价交换，不想单方面付出。这是一种健康的关系模式。⚖️

---

**【性格题-29】你在朋友圈发了一条动态，但点赞和评论很少，你会？**

- **A. 有点失落，但很快就会忘记**  
  → 结果「淡定型」：你其实没那么在意点赞，只是偶尔会有点失落。这很正常，谁也不是完全不在乎。📱

- **B. 开始反思是不是发的内容有问题**  
  → 结果「自省型」：你会从自己身上找原因。但有时候不是内容问题，只是时机或运气问题。🔍

- **C. 删掉动态，当作什么都没发生**  
  → 结果「敏感型」：你很在意自己在别人眼里的形象。但别忘了，发朋友圈是为了自己开心，不是为了表演。🗑️

- **D. 完全不在意，爱点赞不点赞**  
  → 结果「洒脱型」：你是真的不在乎这些虚的东西。朋友圈只是记录生活，点赞数说明不了什么。🌟

---

**【性格题-30】班级/社团聚餐有人提议玩真心话大冒险，你会？**

- **A. 积极参与，趁机了解一下平时不熟的同学**  
  → 结果「社交型」：你觉得这是拉近关系的好机会。玩得起也放得开，是聚餐的气氛担当。🎉

- **B. 低调观察，看别人玩自己跟着乐**  
  → 结果「观察型」：你喜欢在场但不抢风头。静静看着也能融入，而且能观察到很多有趣的细节。👀

- **C. 找借口提前离开，实在不想参与**  
  → 结果「回避型」：你对这种游戏有压力，觉得被动曝光隐私很不自在。不去也没关系，不必勉强自己。🚪

- **D. 参加但提前想好"标准答案"应付各种问题**  
  → 结果「策略型」：你会做准备，不打无准备的仗。既想参与又不想翻车，这是你的生存智慧。🧠

### 2.2 情感测试题（20题）

> 特点：暗恋/恋爱/分手/室友关系等学生情感场景

#### 情感题 1-5：暗恋篇

---

**【情感题-01】你暗恋一个人很久了，最近发现ta好像注意到你了，你会？**

- **A. 更加主动，创造更多接触机会**  
  → 结果「勇敢追型」：你信奉"喜欢的人就要追到手"。等对方主动太慢，不如自己掌握主动权。🏹

- **B. 保持现状，让关系自然发展**  
  → 结果「顺其自然型」：你觉得感情急不得，顺其自然才是最好的节奏。给对方空间也给自己退路。🌱

- **C. 开始患得患失，反而不知道该怎么相处了**  
  → 结果「纠结型」：得到一点希望就开始患得患失，这是暗恋者最常见的心理。但小心太紧张反而弄巧成拙。😰

- **D. 找朋友帮忙分析ta到底是什么意思**  
  → 结果「求助型」：你懂得借助外力判断局势。但朋友的分析不一定准，最终还是要自己决定。👥

---

**【情感题-02】暗恋对象经常找你聊天，但从来不主动约你，你会怎么理解？**

- **A. ta可能只是把我当普通朋友**  
  → 结果「理性分析型」：你会冷静分析局势，不盲目乐观。但有时候也需要一点冲动，万一呢？🤔

- **B. ta可能在试探我，我应该更主动一点**  
  → 结果「主动出击型」：你相信"爱情靠争取"。与其猜来猜去，不如主动一点试探对方反应。⚡

- **C. 算了，聊聊天也很好，不强求更多**  
  → 结果「知足常乐型」：你觉得能聊天已经很好了，不敢奢求太多。但有时候主动一下会有惊喜哦。🌙

- **D. 直接问ta到底什么意思，不想再猜了**  
  → 结果「直球型」：你是那种"与其内耗不如摊牌"的人。问清楚虽然尴尬，但比一直猜来猜去强。💬

---

**【情感题-03】你终于鼓起勇气向暗恋对象表白，但被发好人卡，你会？**

- **A. 笑着说"没关系，做朋友也很好"然后默默走开**  
  → 结果「体面型」：你懂得给彼此留台阶。表白本身就需要勇气，被拒绝了也不丢人。🌸

- **B. 很伤心，想躲起来一个人待几天**  
  → 结果「自我疗愈型」：你需要时间消化情绪，这很正常。允许自己难过，但别沉溺太久哦。💔

- **C. 问清楚原因，想知道自己哪里可以改进**  
  → 结果「成长型」：你把拒绝当成学习机会。但有时候不喜欢就是不喜欢，跟你好不好没关系。📈

- **D. 删掉ta的联系方式，决定再也不联系**  
  → 结果「断舍离型」：你选择彻底告别，不给自己留念想。这是对自己的保护，但可能也有点极端。🗑️

---

**【情感题-04】你和暗恋的人被共同朋友看出来了，朋友要帮你撮合，你会？**

- **A. 拒绝，觉得这样逼出来的很没意思**  
  → 结果「自尊型」：你希望感情是自然而然的，不喜欢被人推着走。被撮合总觉得变了味。🚫

- **B. 同意，让朋友帮忙探探对方口风**  
  → 结果「借助外力型」：你觉得朋友是很好的桥梁，能帮你减少误判。有人在中间传话比自己瞎猜强。🤝

- **C. 半推半就，看朋友打算怎么做再说**  
  → 结果「观望型」：你不想主动但也不想拒绝机会。让朋友先探路，自己再决定下一步。👀

- **D. 反过来求朋友保密，不想让事情变得更复杂**  
  → 结果「隐藏型」：你还没准备好面对，宁可维持现状。但有时候错过就是因为太保守。🤐

---

**【情感题-05】暗恋很久的人突然脱单了，对象不是你，你会？**

- **A. 真心祝福，不想让自己显得小气**  
  → 结果「大度型」：你能祝福曾经的暗恋对象，这是一种成熟。放下也是一种成全。🌈

- **B. 默默退出ta的世界，减少接触**  
  → 结果「自我保护型」：你知道继续关注只会让自己更难受。最好的放下是不再关注。🚶

- **C. 偷偷难过很久，但表面上装作若无其事**  
  → 结果「隐忍型」：你把情绪藏得很深，不想让别人看到你的脆弱。但长期压抑对自己不好哦。😢

- **D. 化悲愤为食欲/购物欲，疯狂犒劳自己**  
  → 结果「自愈型」：你懂的发泄方式很重要。与其憋着不如让自己爽一把，吃饱喝足再说。🛍️

- **C. 教室/自习室（安静无干扰）**  
  → 结果「效率型」：你喜欢纯粹的学习环境。安静、没人、只有书和题，这就是你的Dream Place。🏫

- **D. 随便哪里，站着蹲着都能学**  
  → 结果「随遇而安型」：你对环境不挑剔，有书就行。真正的学霸是不挑场地的。🌿

---

**【学习风格-07】你更倾向于用什么方式整理笔记？**

- **A. 抄书/抄PPT，完整记录**  
  → 结果「记录型」：你是那种"好记性不如烂笔头"的信徒。笔记越全越有安全感。📓

- **B. 只记重点和关键词**  
  → 结果「精简型」：你觉得笔记是用来提醒自己的，不是用来抄书的。能看懂就行，不需要完美。✏️

- **C. 做思维导图，把知识点串联起来**  
  → 结果「结构型」：你喜欢从全局看知识，一张图胜过千百字。知识点之间的联系比单个知识点更重要。🕸️

- **D. 不做笔记，觉得没用**  
  → 结果「理解型」：你觉得上课认真听懂了就行，笔记只是安慰剂。真正理解了的东西不需要记录。🧠

---

**【学习风格-08】成绩出来后考砸了，你的第一反应是？**

- **A. 难过一会儿，然后分析原因**  
  → 结果「复盘型」：你能控制情绪，失败不可怕，可怕的是不知道为什么失败。分析清楚才能下次进步。📊

- **B. 找老师要分，觉得改错了**  
  → 结果「争取型」：你觉得自己的权益要自己争取。万一真的改错了呢，不问永远没机会。💪

- **C. 默默接受，告诉自己下次努力**  
  → 结果「内敛型」：你不喜欢申诉，觉得考差了是自己没学好。暗自下决心下次一定要翻身。📚

- **D. 转移注意力，先不想考试的事**  
  → 结果「逃避型」：你选择暂时不看不想，让情绪自然消化。逃避虽然可耻但有点用。🌊

## 五、文案风格指南

### 5.1 整体语调

| 场景 | 语调 |
|------|------|
| 结果描述 | 温暖、正向、具体，避免"你是一个很普通的人" |
| 引导分享 | 轻快、有趣、引导行动 |
| 情感测试 | 细腻、浪漫、有代入感 |
| 趣味测试 | 轻松、幽默、接地气 |
| 学习测试 | 鼓励、支持、提供方法 |

### 5.2 禁用词汇

```
❌ 不准说："你可能..."
❌ 不准说："大概率"
❌ 不准说："也许"
❌ 不准说："其实你很一般"
❌ 不准说："你不应该..."

✅ 应该用："你通常会..."
✅ 应该用："你是...类型"
✅ 应该用："这很符合你"
✅ 应该用："你的特点是..."
```

### 5.3 结果描述公式

```
结果 = 具体行为描述 + 正面肯定 + 专属感 + 可分享性

示例：
"你是一个'深夜卧谈爱好者'——和室友聊到凌晨三点是你的常规操作。
你珍惜这种能说知心话的时光，朋友眼中你是最好的倾听者。
快把这个标签发给室友，看看ta是不是同款！"
```
